## v2.0.0 — Professional Architecture, Engineering & Preconstruction Platform

Budget Engineer v2.0.0 is the first release where the platform becomes a coherent **local-first architecture, engineering, preconstruction, procurement, and delivery system**.

This release combines:

- architecture-aware prompt-to-plan generation
- canonical room, wall, and opening geometry
- multi-storey and typology-specific planning
- technical drawing and sheet intelligence
- trade-detailed early-stage BOQ, schedules, programme, and cashflow
- site analysis and heliodon workflows
- interior documentation foundations
- structural and MEP pre-design
- assurance, delivery, procurement, handover, and project-controls lifecycle workflows
- strong local-first export/import continuity

## Product Positioning

Budget Engineer v2.0.0 is best described as a:

**professional-grade, local-first architecture, engineering, and preconstruction platform with human review and signoff in the loop**

It should **not** be described as:
- a full licensed architect replacement
- final structural signoff software
- automatic certified code approval
- a native proprietary BIM/CAD suite replacement

---

## Why v2.0.0 Matters

This release marks the transition from:
- a strong architecture-generation and BOQ platform

to:
- a broader professional system that now connects design, documentation, cost, schedule, assurance, delivery, and handover workflows.

That makes Budget Engineer far more credible as a real professional platform for:
- architectural planning
- preconstruction
- procurement-linked coordination
- issue-driven project progression
- local-first project continuity
