## Overview

**Budget Engineer v2.0.0** is the first release where the system becomes more than a design-costing tool.

It now operates as a broader **local-first architectural planning, documentation, engineering, and preconstruction platform**, combining:

- architecture-aware plan generation
- validated canonical geometry
- multi-storey and typology-specific planning
- technical drawing and sheet intelligence
- trade-detailed BOQ and schedules
- programme / Gantt / cashflow
- site analysis
- interior documentation
- structural and MEP pre-design
- review workflows
- delivery packaging

This release is the point where Budget Engineer becomes credible not just as an experimental generator, but as a serious **professional platform with human review in the loop**.

## What's New in v2.0.0

### 1. Architectural Generator Intelligence

The generator now supports far stronger architectural logic than earlier versions.

**Key improvements include**
- constrained zoning
- smart openings
- adjacency-driven wall logic
- typology-aware planning
- vertical chassis and multi-storey organization
- floor-role-aware generation
- unit-aware apartment modelling
- mixed-use entrance separation
- canonical geometry normalization
- canonical wall graph
- validated opening host recovery
- rejection of invalid required-room layouts

**Result**
The generator is now significantly more capable across:
- detached houses
- villas
- duplexes
- apartment blocks
- mixed-use buildings
- clinics
- schools
- warehouse + office
- worship/community halls

## Known Limitations

The platform is significantly stronger, but some limitations still remain:

### Professional responsibility
- licensed human review and signoff are still required

### Structural / MEP depth
- structural and MEP outputs remain pre-design / review-oriented, not final engineered signoff documents

### Code / review
- code support is review-assistive, not certified authority approval

### Typology maturity
- some complex typologies may still benefit from further hardening and calibration

### Validation
- benchmark and calibration depth should continue to expand as more real project references are added
