# Capability Highlights

## Design Pipeline

| Stage | What Happens | Demo Evidence |
|-------|-------------|---------------|
| Brief input | Plain English → parsed building parameters | 3-bed house, 120m², Harare |
| Design generation | 3 options with varied dimensions | Standard / Compact / Spacious |
| Plan model | 9 rooms, 11 walls, 10 openings | Floor plan with room labels |
| CAD document | 2D CAD with layers, walls, openings | Editable in CAD panel |
| 3D BIM viewer | IFC-style classification, camera controls | 3D tab in project dashboard |

## Cost Estimation

- 14-line-item BOQ with realistic Harare rates
- 4 sections: Substructure, Superstructure, Finishes, MEP
- Total ~$42K with 10% contingency
- Region-aware pricing (Zimbabwe rates seeded from database)
- CSV and PDF export available
